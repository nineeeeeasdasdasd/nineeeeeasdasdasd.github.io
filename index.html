รางวัลที่ภาคภูมิใจ

ผลงานเรื่องไลน์บอทตัวจับโรคมะเขือเทศด้วยปัญญาประดิษฐ์

ได้รับการคัดเลือกเข้าร่วมประกวดรางวัลนวัตกรรมแห่งประเทศไทย ครั้งที่ 23 เนื่องในงานมหกรรมวิทยาศาสตร์และเทคโนโลยีแห่งชาติ ประจำปี 2566

ผลงานเครื่องให้อาหารไก่กึ่งอัตโนมัติ

ได้รับรางวัลระดับเหรียญเงิน กิจกรรม การประกวดโครงงานวิทยาศาสตร์ ประเภทสิ่งประดิษฐ์ งานศิลปหัตถกรรมนักเรียน ระดับชาติ ครั้งที่ 70

การแข่งขันการสร้าง Web Applications ครั้งแรกในชีวิต

ได้รับรางวัลเหรียญทอง รองชนะเลิศอันดับที่ 1 กิจกรรมการแข่งขันการสร้าง Web Applications ในงานศิลปหัตถกรรมนักเรียน ระดับเขตพื้นที่การศึกษา ครั้งที่ 70
